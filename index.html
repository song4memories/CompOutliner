Comp Outliner — Support

English

Comp Outliner is an outliner for long-form writing — memos, articles, papers, novels, and screenplays — with live preview and genre-specific tools.

Contact

Questions, bug reports, and requests: song4memories@gmail.com

FAQ

What file format does it use? Documents are saved as .comp documents; double-click one in Finder to open it. Large documents switch (after asking you) to a package format with the same .comp extension, in which each node's body is stored as its own plain-text file. You can also export to Markdown, OPML, DOCX, EPUB 3, and PDF.

How do I add nodes quickly? Press ⌘↩ for a sibling and ⇧⌘↩ for a child. While typing a title, ↩ creates the next node and Tab indents; press ↩ or Esc on an empty title to stop.

Where are the Novel and Screenplay genres? They appear in the genre menu (top of the tree pane) when Show English writing genres is enabled in Settings ▸ General. It is enabled by default unless your system language is Japanese.

Can I import or export screenplays? Yes — Fountain and Final Draft (FDX). Import with File ▸ Import Screenplay…; export from the screenplay export formats, alongside the US screenplay PDF.

Related apps

For vertical Japanese novel and screenplay writing, see Comp Writer (Japan). For bilingual translation, see Comp Translator.

Privacy

日本語

Comp Outliner は長文執筆のためのアウトラインプロセッサです。メモ・記事・ 論文（と英語の小説・脚本）に対応し、ライブプレビューとジャンル別ツールを 備えます。

よくある質問

ファイル形式は？ 文書は .comp 書類として 保存し、Finder でダブルクリックすると開きます。大きな文書は確認のうえ、 同じ .comp 拡張子のまま各ノードの本文を個別のテキストファイル として内包するパッケージ形式に切り替わります。Markdown / OPML / DOCX / EPUB 3 / PDF への書き出しにも対応しています。

ノードを素早く追加するには？ ⌘↩ で兄弟、 ⇧⌘↩ で子を追加します。タイトル入力中は ↩ で次の ノードを作成、Tab で字下げ。空のまま ↩ か Esc で終了します。

小説・脚本ジャンルが見当たりません。 設定 ▸ 一般の 「英語の執筆ジャンル (小説 / シナリオ) を表示」をオンにすると、ジャンル メニューに米国書式の小説とシナリオが追加されます（システム言語が日本語の 場合は既定でオフ）。

脚本の取り込み・書き出しは？ Fountain と Final Draft (FDX) に対応しています。取り込みはファイル ▸ Import Screenplay…、書き出しは シナリオ書き出しの形式（US スクリーンプレイ PDF と並び）から選べます。

関連アプリ
